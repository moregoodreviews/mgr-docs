# Customization Options

We offer several ways you can customize your client portal beyond the basics.

## Page Title, Description, & Preview Image

You can edit the meta title, description, and preview image (shown when sharing the link to your client portal) in Agency > Appearance. These settings override the current settings of the platform.

## External Links

You can add additional links to the sidebar of your client portal in Agency > External Links. These links will be shown below the existing sections in the sidebar. This allows you to link out to your own docs, billing, or sales materials.

## Styles & Scripts

If you are a bit savvy, you can customize the CSS of the client portal, or add JavaScript tags to 3rd party services in Agency > Code Editor. This gives you complete control over the look and feel of the platform, as well as allow you to add your own help desk software, 3rd party analytics, and more.

## Page Footer

You can set the contents of the footer of each page in the client portal in Agency > Settings. This allows you to add your own disclaimers, legal info, or ancillary links to the platform. On this page, you may also disable our own cookie consent banner. If you would like to use your own cookie consent service, you could add an embed code in the Scripts section of the Code Editor page.
