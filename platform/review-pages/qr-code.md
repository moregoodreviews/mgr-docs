# QR Code

You can download a QR code that links directly to your review page. Place it on receipts, in-store signage, tables, or anywhere customers can scan it to leave feedback. When they scan, they go straight to your review page.

---

### Where to Find It

Open a review page and click **QR Code** in the sidebar. Configure the options and click **Download** to get your QR code file.

---

### Options

* **Color** – The color of the QR code. By default, it uses your project's primary color. Choose a color that contrasts with the background where you'll print or display it.

* **Location** – If you've [assigned locations](locations.md) to this review page, you can select a location for the QR code. The code will link to the review page with that location pre-selected, so reviews from that scan are attributed to that location. Useful when you have different QR codes at different stores.

* **Tags** – Optionally assign tags to reviews that come from this QR code. Helps you segment or filter reviews in reports.

* **Output** – Choose the file format:
  * **SVG** – Vector format. Scales without losing quality. Best for print and design work.
  * **PNG** – Raster format. Good for web and simple printing.

---

### Using Your QR Code

After downloading, you can:

* Print it on receipts, flyers, or table tents
* Add it to in-store signage or window decals
* Include it in email signatures or marketing materials
* Display it on a tablet or screen for customers to scan

{% hint style="info" %}
For best results, print the QR code at least 2 cm (about 0.8 inches) square so it's easy to scan with a phone camera.
{% endhint %}

---

### Tips

{% hint style="success" %}
Use location-specific QR codes when you have multiple stores. Each location gets its own code, and reviews are automatically attributed to the right place.
{% endhint %}

{% hint style="info" %}
Test the QR code after downloading. Scan it with your phone to confirm it goes to the correct page and that the location (if set) is pre-selected.
{% endhint %}
